# Referencias de diseño

Aquí van los mockups y exports que salgan de Claude Design u otras fuentes.

- `mockups/` → imágenes de pantallas (PNG/JPG). Nombra por pantalla y versión:
  `editor-v1.png`, `home-v1.png`, `library-v1.png`, `tray-v1.png`.
- `exports/` → cualquier export adicional (SVG, paletas, código del diseño, etc.).

Cuando subas los mockups, avísame y los reviso para extraer tokens reales
(colores, tipografías, sombras) hacia `src/core/theme` y actualizar
[../05-design-system.md](../05-design-system.md).
