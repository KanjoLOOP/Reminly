## Estado

MVP funcional: estantería de libretas, editor de lienzo con fotos/GIF/vídeo/voz/texto/
stickers/washi, personalización completa, persistencia local, bandeja de recuerdos y backup.
Privacidad por diseño (local-first, sin red, EXIF eliminado).

Siguiente: preparar la publicación en Play Store (icono, splash, build con EAS).
